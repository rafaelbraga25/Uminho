O ficheiro cp1415t.lhs representa o programa deste projeto.
No ficheiro está contido o esquelo do relatório em latex, bem como o 
código em haskell do programa. O ficheiro possui também alguns 
exemplos de teste que podem ser executados através do programa ghci

Relatório: cp1415t.pdf
pyr.html: resultado da execução da função drawPyr
triangle.html: resultado da execução da função draw

Requisitos: 
 - ghci
 - haddock
 - cabal: sudo apt-get install cabal-install
 - System.Random: cabal install random
 - Control.Parallel.Strategies: cabal install parallel
